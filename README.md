# Prism

Prism is a Vite, React, and TypeScript landing page for a visual arts and storytelling collective.

Live site: https://prism-eta-one.vercel.app

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
